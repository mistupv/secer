# Core Erlang Constraints


### Guards


* (T) True Guard

* (F) False Guard


### Matching Literals


* (T) Match Equal

* (F) Match Not Equal


### Matching Tuples


* (T) Tuple of Size N

* (F) Tuple of Not Size N

* (F) Not a Tuple


### Matching Lists


* (T) Nonempty List

* (F) Empty List

* (F) Not a List
